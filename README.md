# Проект: Сайт для дилерской компании LADA

## Описание проекта

Данный сайт разработан в рамках индивидуального итогового проекта.  
Тема: **«Создание сайта для дилерской компании»** (на примере бренда LADA).

Сайт предназначен для дилерского центра, реализующего автомобили LADA.  
Он помогает привлекать клиентов, предоставлять информацию об автомобилях и собирать заявки.

## Функционал сайта

- Главная страница с акциями, преимуществами и карточками моделей
- 5 отдельных страниц для моделей: Granta, Vesta, Niva 4x4, Niva Travel, Largus
- Таблицы технических характеристик
- Вкладки (табы) для переключения между разделами:  
  «Характеристики», «Преимущества», «Слабые места», «Отзывы»
- Формы обратной связи (тест-драйв, кредит, обратный звонок)
- Адаптивный дизайн (корректно отображается на компьютерах, планшетах и телефонах)

## Использованные технологии

- HTML5
- CSS3
- Bootstrap 5
- JavaScript (валидация форм)
- GitHub Pages (хостинг)

## Структура сайта

- `index.html` – главная страница
- `granta.html`, `vesta.html`, `niva.html`, `niva-new.html`, `largus.html` – страницы моделей
